# 竞价开盘标准

| 形态               | 次日正常/符合预期       | 次日不及预期       | 次日超预期         |
|--------------------|------------------------|------------------|------------------|
| 午后跌停           | 低开1-3个点            | 低开超过3个点    | 高开带量         |
| 早盘快速跌停       | 低开5-7个点            | 低开超过7个点    | 高开带量         |
| 跌停当日缩地量无量能 | 开一字跌停板低开10个点 |                 | 高开带量         |
| 无利好利空普通形态 | 平开/小低开/小高开     | 低开超过3个点    | 高开3%以上带量   |
| 炸板               | 低开≤2%                | 低开超过4%       | 高开带量         |
| 地天板             | 低开1-3%               | 低开超过4%       | 高开带量         |
| 一字板/秒板        | 低开1-4%               | 高低开不超4%     | 无量一字板       |
| 上午涨停           | 高开4-6%               | 高低开不超3%     | 高开≥7%或一字板  |
| 下午涨停           | 高开3-5%               | 低开超过3%       | 高开≥5%（越强越好）|

## 核心口诀
▸ 当强不强则为弱，当弱不弱则为强

▸ 该涨不涨必有大跌，该跌不跌必有大涨

▸ 弱势出大牛，烂板出妖股  

▸ 符合预期持有，低于预期止损，超预期加仓
